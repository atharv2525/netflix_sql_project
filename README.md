# Netflix Data Analysis using SQL

-> This project analyzes the Netflix Titles Dataset using SQL to extract meaningful business insights.
  The dataset contains information about movies and TV shows available on Netflix.

The goal of this project is to practice and demonstrate SQL skills including:

Data filtering,Aggregation,Grouping,Window functions,String manipulation,Case statement,sDate functions,Data cleaning.

Key Insights:

->Movies dominate Netflix compared to TV Shows.

->Certain ratings are significantly more common.

->India has shown rapid growth in Netflix content over recent years.

->A small number of actors appear frequently in Indian productions.

->Keyword-based classification shows majority content falls under “Good” category.
